# 获客精灵 — 页面原型与交互规格

## 页面清单

| 页面 | 路径 | 权限 | 优先级 |
|------|------|------|--------|
| 登录 | /login | 公开 | P0 |
| 注册 | /register | 公开 | P0 |
| 仪表盘 | /dashboard | 登录 | P0 |
| 视频监控 | /dashboard/videos | 登录 | P0 |
| 评论列表 | /dashboard/comments | 登录 | P0 |
| 话术管理 | /dashboard/templates | 登录 | P1 |
| 数据报表 | /dashboard/analytics | 登录 | P1 |
| 设置 | /dashboard/settings | 登录 | P2 |
| 套餐升级 | /pricing | 公开 | P1 |

```

│          获客精灵 Logo              │
│    社交媒体智能截流与私域引流工具      │

│  │ 邮箱                         │  │

│  │                             │  │
│  │ [        登 录             ] │  │
│  │                             │  │
```

### 交互
- 输入邮箱/密码 → 点击登录 → 加载状态 → 成功跳转仪表盘 / 失败显示错误
- 错误提示：红色边框 + 下方红色文字
- 加载状态：按钮变灰 + "登录中..." + 禁用输入

## 10. 交互流程图

### 添加监控流程
```
用户点击 [+ 添加监控]
    ↓
弹出"添加视频监控"弹窗
    ↓
用户输入视频链接 + 选择平台
    ↓
用户点击 [开始监控]
    ↓
系统验证链接格式
    ↓
系统开始抓取评论（异步）
    ↓
弹窗关闭，显示 Toast "监控已添加"
    ↓
视频列表新增一条记录（状态：抓取中）
    ↓
30秒后自动刷新，显示抓取到的评论数
```

### 处理高意向用户流程
```
系统识别到高意向评论
    ↓
浏览器通知 + 仪表盘红点提醒
    ↓
用户点击 [快速回复]
    ↓
弹出"回复用户"弹窗
    ↓
用户选择话术或自定义回复
    ↓
用户点击 [发送回复]
    ↓
系统检查频率限制
    ↓
发送回复（随机延迟 30-120 秒）
    ↓
更新评论状态为"已回复"
    ↓
Toast "回复已发送"
    ↓
询问是否继续发送私信？
    ↓
用户选择发送私信
    ↓
系统检查私信频率限制
    ↓
发送私信（随机延迟 1-3 分钟）
    ↓
更新状态为"已私信"
```
